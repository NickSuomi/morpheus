## Brand idea

Morpheus is the control plane between free agentic exploration and evidence-backed repository work.

Core motif: mirrored portal / hourglass mark.

Meaning:

- left side: dream, exploration, possibility;
- center pinch: operator control / intent boundary;
- right side: evidence, review artifacts, merge authority.

## Copy

Primary tagline:

> Dream with no limits. Run with evidence.

Short description:

> Agent ops for operators running AI work on real repositories.

Launch headline:

> From intent to impact. Under human authority.

## Principles

1. Explicit Intent
2. Durable State
3. Sandboxed Execution
4. Transcripts & Logs

## Usage notes

- Use `morpheus-logo-mark.svg` on light backgrounds.
- Use `morpheus-logo-mark-reversed.svg` on dark backgrounds.
- Use `morpheus-logo-lockup.svg` for README/header contexts.
- Use `morpheus-og-image.png` for GitHub/social preview.
- Use `morpheus-hero-flow.svg` in README hero section or docs landing page.
- Use `morpheus-evidence-flow.svg` as the README compatibility alias for the
  hero flow artwork.
- Keep generous whitespace. Do not place mark inside busy AI-style backgrounds.
